Funções
- Receber alertas: Listar todos os alertas enviados pela plataforma 
android do UrbanFix (UrbanFix Mobile)
- Atualizar status: Listar detalhe dos alertas e alterar o status deste 
a fim de enviar informação de feedback para o UrbanFix Mobile
- Gerenciar funcionario: Listar detalhe do usuário do UrbanFix Web 
(funcionário) e editar seus dados se for necessário
- Emitir Relatórios: gerar e exibir relatórios através diversos gráficos 
baseados nos dados dos alertas

Possíveis futuras funções
- Cadastro de funcionários: Criar um super usuário para gerenciar todos 
us funcionários a fim de efetuar funções CRUD (cadastro, listagem, 
edição e exclusão), com prioridade em cadastro de novos usuários, a fim 
de evitar o cadastro destes direto no banco de dados
- Envio de email: Não existe forma de comunicação entre o órgão 
responsável por tratar o problema e o sistema. Será necessário efetuar 
cadastro de órgãos públicos a fim de ter contatos cadastrados (email e 
telefone) e através deste cadastro, será possível criar um sistema de 
envio de email. Assim o sistema se completa em sua função de encaminhar 
os alertas aos órgãos responsáveis pelo tratamento do alerta
